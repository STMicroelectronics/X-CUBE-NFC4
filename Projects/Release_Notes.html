Purpose

The X-CUBE-NFC4 Projects provide examples and applications to aid understanding how to develop an application using the NFC Middleware for the ST25DV device. Projects include STM32CubeMX project file (.ioc) for the graphical visualization of the STM32 microcontroller pins, peripherals and middlewares configuration.
